## Pow

`func Pow(x, y float64) float64`

> 第一个float64指参数的类型，第二个float64指返回的类型。

Pow returns x**y, the base-x <a title="指数">exponential</a> of y.

Special cases are (in order):

```Go
Pow(x, ±0) = 1 // for any x
Pow(1, y) = 1 // for any y
Pow(x, 1) = x  // for any x

.
.
.

<++>
```
